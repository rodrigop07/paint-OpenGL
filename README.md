# paint-OpenGL

Funcionalidades:
- Criação de pontos
- Criação de retas
- Criação de polígonos
- Seleção de objetos
- Exclusão de objetos selecionados
- Transladação de objetos selecionados
- Rotação de objetos selecionados
- Escalonamento de objetos selecionados



Aperte `v` para criar pontos  
Aperte `r` para criar retas  
Aperte `p` para criar polígonos  
Aperte `s` para selecionar objetos  
Aperte `d` no modo de seleção para excluir objetos selecionados  
Aperte o `botão esquerdo` do mouse para adicionar vértices  
Aperte o `botão direito` do mouse para finalizar um polígono   
Aperte `Shift` + `<-` para rotacionar objetos selecionados para a esquerda  
Aperte `Shift` + `->` para rotacionar objetos selecionados para a direita  
Aperte `Shift` + `^` para escalonar objetos selecionados para cima  
Aperte `Shift` + `v` para escalonar objetos selecionados para baixo  
Aperte `Ctrl` + `<-` para mover objetos selecionados para a esquerda  
Aperte `Ctrl` + `->` para mover objetos selecionados para a direita  
Aperte `Ctrl` + `^` para mover objetos selecionados para cima  
Aperte `Ctrl` + `v` para mover objetos selecionados para baixo  
Aperte `Alt` + `<-` ou `Alt` + `->` para espelhar objetos selecionados no eixo vertical
Aperte `Alt` + `^` ou `Alt` + `v` para espelhar objetos selecionados no eixo horizontal
Aperte `Shift` + `a` para cisalhar objetos selecionados na direção x negativamente  
Aperte `Shift` + `s` para cisalhar objetos selecionados na direção x positivamente  
Aperte `Shift` + `z` para cisalhar objetos selecionados na direção y negativamente  
Aperte `Shift` + `x` para cisalhar objetos selecionados na direção y positivamente
