# paint-OpenGL

Funcionalidades:
- criação de pontos
- criação de retas
- criação de polígonos

Aperte 'q' para criar pontos
Aperte 'w' para criar retas
Aperte 'e' para criar polígonos
Aperte o botão esquerdo do mouse para adicionar vértices
Aperte o botão direito do mouse para finalizar um polígono
